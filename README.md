# Glowbal V2

Serious product build for Glowbal.

## Stack
- Next.js
- Supabase
- Tailwind CSS

## Initial focus
- auth
- onboarding
- student profile
- optional CV upload
- shortlist results
- save and compare

## Environment variables
Copy `.env.example` to `.env.local` and fill in:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`

## Run locally
```bash
npm install
npm run dev
```
